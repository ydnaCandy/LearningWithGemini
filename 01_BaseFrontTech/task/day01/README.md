# Day 01: HTMLの基本構造と基本タグ

## 学習目標
- HTMLファイルの基本的な書き方を知る
- `h1`, `p`, `a` タグを使えるようになる

## 課題
1. この `day01` フォルダを `01_BaseFrontTech/learn/` にコピーしてください。
2. `learn/day01/` 内に `index.html` を作成してください。
3. `index.html` に以下の内容を含めてください。
    - 見出し（`h1`）: 「自己紹介」
    - 段落（`p`）: 好きなことや趣味を1文で書いてください。
    - リンク（`a`）: 好きなWebサイトへのリンク（例: Google, YouTube等）
4. 完成したら、Geminiに「Day 01の課題が終わりました」と報告してください。

## ヒント
HTMLの基本テンプレート：
```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>Day 01 Task</title>
</head>
<body>
    <!-- ここにコードを書いていく -->
</body>
</html>
```
