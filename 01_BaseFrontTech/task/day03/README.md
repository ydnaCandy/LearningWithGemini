# Day 03: divによるグループ分け

## 学習目標
- `div` タグを使ってコンテンツをグループ化する意味を理解する

## 課題
1. `task/day03` フォルダを `learn/` にコピーしてください。
2. `index.html` を作成し、2つの「カード」を作成してください。
3. 各カードは `div` タグで囲み、その中に「見出し」と「説明文」を入れてください。
    - カード1: タイトル「HTML」、説明「Webの骨組みを作る」
    - カード2: タイトル「CSS」、説明「Webのデザインを整える」
4. 完成したら報告してください。

## なぜdivを使うのか？
`div` 自体には特別な見た目の意味はありませんが、後の学習で「このグループだけ背景色を変える」「横並びにする」といったデザイン（CSS）を適用するために非常に重要になります。
