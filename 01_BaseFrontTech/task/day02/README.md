# Day 02: いろいろなタグ

## 学習目標
- 画像を表示するタグ（`img`）の使い方を学ぶ
- リスト（`ul`, `li`）の作り方を学ぶ
- 入力フォーム（`input`）の基本を知る

## 課題
1. `task/day02` フォルダを `learn/` にコピーしてください。
2. `index.html` を作成し、以下の要素を追加してください。
    - リスト: 好きな食べ物を3つリスト（`ul`, `li`）で表示。
    - 画像: 何か好きな画像を表示（ネット上の画像のURLを `src` に指定してください）。
    - 入力欄: 自分の名前を入力できる `input` タグ（`type="text"`）。
3. 完成したら報告してください。

## ヒント
```html
<img src="画像のURL" alt="画像の説明">
<ul>
    <li>項目1</li>
</ul>
<input type="text" placeholder="ヒントテキスト">
```
